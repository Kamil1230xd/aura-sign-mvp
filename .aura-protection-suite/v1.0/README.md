# AURA Protection Suite v1.0

This directory contains the AURA Protection Suite v1.0 artifacts.

## Components

- **security-policy.md** - Security policy and vulnerability reporting guidelines
- **signing-config.json** - Configuration for commit signing and verification
- **protection-rules.yml** - Branch protection and code review rules

## Purpose

The AURA Protection Suite provides enhanced security and integrity measures for the repository, including:

- Commit signature verification
- Automated security scanning
- Code ownership enforcement
- Branch protection policies
